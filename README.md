Ace-I-Kernel
============

Kernel for Galaxy Ace-i based on Rafael.Baugis (bieltv3 actually) kernel sources.

Minor changes:

Frequencies:
- 832 (1.27V)
- 748 (1.26V)
- 624 (1.24V)
- 532 (1.23V)
- 468 (1.22V)
- 312 (1.20V)
- 208 (1.17V)

Governors:
- ondemandx
- lagfree
- conservative
- interactive
- brazilianwax
- interactiveX
- ondemand
- powersave
- Lionheart
- smartassV2 (default)
- performance
- zzmoove
- and... of course bcm21553 epic governor!

I/O schedulers:
- noop (default)
- deadline
- cfq
- vr
- sio
- bfq

and some changes.
